 
# ExploreID

**ExploreID** merupakan platform berbasis web yang dirancang untuk membantu wisatawan memperoleh informasi destinasi wisata secara lengkap serta melakukan reservasi tiket secara online. Sistem ini menyediakan informasi destinasi, fasilitas, lokasi, harga tiket, ulasan pengunjung, serta fitur pemesanan dan e-ticket untuk mempermudah proses kunjungan wisata.

## Fitur Utama

| Modul | Fitur |
|---|---|
| Dashboard | Statistik ringkasan, grafik, akses cepat |
| Kelola Destinasi | Tambah, edit, hapus destinasi wisata |
| Kelola Kategori | Manajemen kategori: Pantai, Gunung, Sejarah |
| Kelola Reservasi | Daftar reservasi, filter status, verifikasi |
| Statistik | Grafik pengunjung dan reservasi bulanan |
| Kelola User | Manajemen wisatawan, pengelola, dan admin |
| Kelola Review | Approve/tolak/hapus ulasan wisatawan |
| Kelola Konten | Banner, Event Wisata, Berita, FAQ |
| Laporan & Export | Laporan pendapatan/pengunjung, Export PDF/Excel |

## Teknologi

- React + TypeScript + Vite
- Recharts (grafik interaktif)
- React Router DOM (navigasi SPA)
- Lucide React (ikon)

## Menjalankan Lokal

- Clone Repository : https://github.com/hendi1510/Exploreid.git
- Menjalankan Mesin API
    - cd api
    - npm install
    - npm run start:dev
- Instalasi Backend
    - cd user
    - npm install
- Instalansi Frontend cms
    - cd cms
    - npm install
- Menjalankan Frontend
    - npm run dev

Akses di: http://localhost:5173
