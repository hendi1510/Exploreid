# ExploreID CMS – Admin Panel

Panel administrasi untuk mengelola seluruh konten website wisata ExploreID.

## Fitur Utama

| Modul | Fitur |
|---|---|
| Dashboard | Statistik ringkasan, grafik, akses cepat |
| Kelola Destinasi | Tambah, edit, hapus destinasi wisata |
| Kelola Kategori | Manajemen kategori: Pantai, Gunung, Sejarah |
| Kelola Reservasi | Daftar reservasi, filter status, verifikasi |
| Statistik | Grafik pengunjung dan reservasi bulanan |
| Kelola User | Manajemen wisatawan, pengelola, dan admin |
| Kelola Review | Approve/tolak/hapus ulasan wisatawan |
| Kelola Konten | Banner, Event Wisata, Berita, FAQ |
| Laporan & Export | Laporan pendapatan/pengunjung, Export PDF/Excel |

## Teknologi

- React + TypeScript + Vite
- Recharts (grafik interaktif)
- React Router DOM (navigasi SPA)
- Lucide React (ikon)

## Menjalankan Lokal

cd cms && npm install && npm run dev

Akses di: http://localhost:5173
